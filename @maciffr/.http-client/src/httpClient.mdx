# HttpClient

HttpClient doit être utilisé par les projets front, il sert comme son nom l'indique à faire des requêtes Http.

## Installation

```shell
npm install -S @maciffr/http-client
```

## Utilisation

## Initialisation

```js

import HttpClient from "@maciffr/http-client"

const httpClient = new HttpClient(fetch.bind(window));
```

### Request Builder

Le **builder est immuable**, à chaque fois qu'une **modification** est apporté, c'est un **nouveau builder** qui est initialiser.
Le but est de se prémunir des risques d'effets de bord, vous pouvez vous construire un builder avec des données de bases et le dérivé ensuite à votre convenance.

#### Constructeur

Vous pouvez fournir directement les paramètres via le constructeur.

Dans le cas ou vous voulez mettre une **url de base** à votre requête, il faut le mettre au niveau du constructeur.

```js

import {GenericRequestBuilder} from "@maciffr/http-client"

const request = new GenericRequestBuilder({

});

```
